Every language has its own folder.

## Rust

```shell
cd rust
cargo run --release -- -f ../data/day4.txt -d 4
```

## Node

```shell
cd node
./bin/run -d 7 -f ../data/day7.txt
```

## Haskell

(requires `stack`)

```shell
cd haskell
chmod +x ./1.hs
cat ../data/day1.txt | ./1.hs
```

Day 24 is a proper subfolder so here you can do

```
stack run < path/to/data
```
